Question-3: What is event delegation in the DOM?

Event Delegation হলো একটি parent element er ওপর Event Listener add করা, যাতে তার child এলিমেন্টগুলোর Event handle করা যায়।

Question-4: What is event bubbling in the DOM?

It is a behavior of DOM Event, যেখানে একটি child এলিমেন্টে Event ঘটলে, সেটি parent element পর্যন্ত bubble উঠে যায়।
